# Electron 小练习

## 执行步骤
``` bash
全局安装 Electron 
sudo npm install --g electron

将程序克隆到本地 
git clone git@github.com:1969290646/electron-demo.git

进入目录
cd electron-demo

运行程序
electron .
```
